Apa itu CORS?

CORS (Cross-Origin Resource Sharing) adalah kebijakan keamanan browser yang mengatur apakah sebuah website boleh mengambil data dari domain lain. Ketika DocView mencoba mengambil file dari URL, browser bertanya ke server: "Bolehkah saya mengakses file ini?"

Jika server tidak mengizinkan, browser memblokir request — bukan karena DocView bermasalah, tetapi karena server sumber memang tidak mengizinkan akses lintas domain.

Biasanya berhasil: file di hosting publik seperti GitHub raw content, CDN publik, atau server yang mengaktifkan CORS
Biasanya gagal: Google Drive, Dropbox, OneDrive, atau server tanpa header Access-Control-Allow-Origin
Tips: Jika URL dari cloud storage gagal, download dulu filenya lalu buka via tab "Dari Komputer"
